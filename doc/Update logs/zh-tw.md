4. 修復了在某些情況下啟用功能頁授權載入異常的問題。
5. 修復了 OneNote 2021 相關文件缺失的問題。
6. 修復了點選取得 Office 歷史版本厚會自動選取版本編號的問題。
7. 持續最佳化，解決 Windows 8.1 下的 Insufficient memory to continue the execution of the program 問題。
